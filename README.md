# Keep Money

### vNext (1.0.0)

## How to start

Please install environment in docs: https://reactnative.dev/docs/environment-setup

### Run node server before run application

```
1. `npm run start`
```

### Android

```
1. `npm run android`
```

### iOS

```
1. Go to ios folder and run: `pod install`
2. `npm run ios`
```
